# Halo: The Master Chief Collection

Npm import prefix: `@blamnetwork/blf/mcc/`

These are BLF chunks by the Halo: The Master Chief Collection main menu. The fileshare metadata (_fsm 1.1) chunk for example is appended to Halo: Reach MCC BLF files, though Halo: Reach does not and cannot read or write this chunk, it is managed entirely via the MCC menu.

The MCC menu is not built like Halo games, as such it does not follow Halo's typically tracked/untracked versioning system.
